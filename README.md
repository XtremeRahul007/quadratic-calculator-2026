# Quadratic Calculator
A responsive quadratic calculator featuring automatic light/dark theme switching, a custom color palette, and a scalable CSS structure. Core JavaScript logic is currently being implemented.

## Features
- Automatic light and dark theme switching ✅
- Custom color palette ✅
- Fully responsive design (mobile to 4K) ✅
- Clean and maintainable CSS structure ✅
- Quadratic equation logic ✅
